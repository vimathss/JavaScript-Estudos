# 📚 JavaScript-Estudos

Este repositório contém todos os meus estudos e exercícios em **JavaScript** enquanto aprendo a linguagem e suas principais funcionalidades.  
A ideia aqui é organizar de forma prática e evolutiva tudo que eu fui aprendendo durante o processo de aprendizado — desde conceitos básicos até tópicos mais avançados.

---

## 🚀 Sobre o projeto

Durante meus estudos de JavaScript, eu dividi os conteúdos em módulos por tema. Esses módulos contêm arquivos de código que exemplificam os principais conceitos que eu estava estudando, funcionando como:

✅ Registro dos meus exercícios  
✅ Código de exemplo para consulta futura  
✅ Aprendizado prático com problemas reais  
✅ Registro do progresso ao longo do tempo
